# 功能详解

> 对应任务书：九.4 程序功能简介 / 九.8 程序输出结果说明。
> 本文档逐页说明平台的 9 个功能页面。计算公式见 [计算方法与公式](计算方法与公式.md)，使用流程见 [安装与运行](安装与运行.md)。

## 总体使用流程

```
注册/登录 → 概览页新建项目 → 参数页填写并确认计算（一键级联 板→次梁→主梁）
        → 板/次梁/主梁 计算页查看与微调结果 → 计算书页生成并打印 PDF
        → 存档页保存历史快照（可选）
```

所有数据按用户隔离，800 ms 防抖自动保存，刷新或换设备登录等操作均不会丢失数据。

---

## 1. 登录 / 注册页（Login）

**功能**：用户身份认证入口，登录与注册两种模式链接切换（非 Tab）。

- **注册**：用户名 3–32 位（字母 / 数字 / 下划线）、密码 ≥ 6 位并二次确认；注册成功后**不自动登录**，自动切回登录页并保留用户名、清空密码。
- **登录**：用户名 + 密码；失败统一返回「用户名或密码错误」（不区分用户名是否存在，防枚举）。
- **会话**：登录成功签发 JWT（7 天有效），存入 `localStorage`；登录后跳转至原目标页（支持 `redirect` 参数）。
- **设计**：居中卡片 + 品牌图标 + 圆角阴影，响应式适配移动端。

> 技术细节（bcrypt 哈希、JWT 无状态、401 拦截器自动跳登录）见 [技术架构 §鉴权与隔离](技术架构.md#鉴权与用户隔离)。

## 2. 项目概览页（Overview）

**功能**：管理当前用户的全部设计项目。

- 以响应式卡片网格（auto-fill，最小 220 px）展示所有项目，含项目名、最后更新时间、是否为当前项目（「当前」标签 + 主色边框）。
- **新建项目**：点击「新建」卡片，弹窗输入项目名（校验后创建并打开，跳转参数页）。
- **打开项目**：点击卡片打开该项目，跳转参数页。
- 加载期间显示骨架卡片，无项目时显示空状态引导。

## 3. 设计参数页（Params）★ 核心

**功能**：一处填入全部设计参数，确认后一键驱动板 → 次梁 → 主梁全链计算。

### 三段式表单

**① 结构参数**（13 项必填）：

| 参数 | 说明 |
|------|------|
| L1（m，平行次梁）/ L2（m，平行主梁 / 板跨方向） | 平面尺寸 |
| 板厚 | 含「建议 80~120」合理性提示 |
| 次梁宽 × 高、主梁宽 × 高、柱宽 | 截面尺寸 |
| 板 / 次梁 / 主梁跨数 | 均 ≥ 2 |
| 次梁箍筋直径、主梁箍筋直径 | 斜截面选筋用 |

**② 荷载参数**（5 项必填）：钢筋混凝土重度、水磨石面层、抹灰厚度、抹灰重度、楼面活载。

**③ 材料常量**：C20（fc = 9.6）、Ⅰ级钢（fy = 270）、Ⅱ级钢（fy = 300）、γd = 1.2、γG = 1.05、γQ = 1.2（**固定，只读展示**）。

### 实时辅助

- **派生几何**：实时显示次梁间距（L2 / 板跨数）、次梁跨度（L1 / 次梁跨数）、主梁跨度（L2 / 主梁跨数）。
- **跨数推荐**：按单跨常用范围推荐跨数区间（板 L2/1.7~2.5、次梁 L1/4~6、主梁 L2/5~8）。
- **截面合理性**：板厚 80~120、次梁 h/跨 1/18~1/12 且 b/h 1/3~1/2、主梁 h/跨 1/15~1/10 且 b/h 1/3~1/2，附 ✓/✗ 提示（仅建议不阻断）。

### 级联确认计算

1. 校验 18 项必填字段是否齐全，缺失则高亮提示；
2. 若已有计算结果，弹窗警告「将覆盖已有计算结果，手改值会丢失」；
3. 确认后顺序调用 `calculate('slab')` → `calculate('beam')` → `calculate('main_beam')`，每次先 `saveNow()` 再 `POST /calculate`；
4. 成功后跳转板计算页。

**快捷按钮**：「填入演示参数」一键填入本组方案数据，「清空」重置全部字段。

## 4. 板计算页（Slab）

**功能**：展示并允许微调板的全部计算结果。

- **荷载表**：水磨石、混凝土、抹灰、恒载 / 活载（标准值与设计值）、折算荷载 g′/q′，可编辑。
- **计算跨度**：边跨 l₀、中跨 l₀（可编辑）。
- **内力表**：五控制截面（边跨跨中、第一内支座、中跨跨中…）的弯矩 M 与剪力 V，可编辑。
- **配筋表**：各截面 As（计算值）与**下拉式选筋**——先选直径（Φ6/8/10/12/14），再选间距（200/180/150/120/100/80），实时计算 As 提供（πd²/4 × 1000/s）并**自动实时**刷新状态标签（推荐 / 不足 / 建议复核）。
- **计算简图**：均布荷载多跨连续梁（>5 跨显示五跨等效，中间跨橙色虚线高亮）。
- **内力图**：弯矩图 + 剪力图并列。
- **截面配筋图**：板带 Φd@s 示意。

> 板不做支座边缘弯矩调整、不验算斜截面（板一般由构造确定），与教材一致。

## 5. 次梁计算页（SecondaryBeam）

**功能**：展示次梁全流程结果（含斜截面）。

- **荷载表**：板传来恒载、次梁自重、粉刷、恒 / 活载、折算荷载 g′（= g_d + q_d/4）、q′（= 3q_d/4）。
- **跨度与内力**：边 / 中跨 l₀、l_n；五截面 M、V，**已做支座边缘调整** M′ = |Mc| − b/2·V₀。
- **正截面配筋**：下拉选筋（n × 直径，受梁宽净距约束），跨中 **T 形截面**、支座**矩形截面**，附截面类型标签。
- **斜截面**：最大剪力、Vc（混凝土抗剪承载力 0.7·ft·b·h₀/γd）、Asv/s、推荐箍筋间距、配箍率。
- **计算简图**：均布荷载连续梁；**抵抗弯矩图**：设计弯矩包络 + 各跨 Mu 抗力带（绿色虚线）。
- **内力图**：M、V 图；**截面配筋图**：T 形 / 矩形 + nΦd。

---

## 6. 主梁计算页（MainBeam）

**功能**：展示主梁（集中力为主）全流程结果。

- **集中荷载表**：G（恒载，含次梁传来 + 主梁自重 + 粉刷）、Q（活载），标准值与设计值。
- **内力（控制截面）**：M1（边跨跨中）、M_B（支座，边缘调整后）、M2（中跨跨中）；VA、VB左、VB右 等，**支座边缘调整** V₀ = (G+Q)/2。
- **正截面配筋**：全 T 形（课程简化），n × 直径选筋。
- **斜截面 + 吊筋**：Vc（0.5·ft·b·h₀/γd，集中荷载为主）、箍筋、**吊筋面积** Asb = γd·F/(fyv·sin45°)。
- **计算简图**：3 跨连续梁，每跨三等分两点集中力 + 柱铰支（柱顶铰 + 柱身 + 地面阴影）。

> 主梁内力采用四工况包络（G 全跨、Q 在 1&3 跨、Q 在 2 跨、Q 在 1&2 跨）取最不利，详见 [计算方法与公式 §主梁](计算方法与公式.md#三主梁)。

## 7. 计算书页（Report）★ 加分项

**功能**：自动装配八章节计算书，A4 预览，浏览器原生打印为 PDF。

- **封面信息表单**：学院、专业、姓名、学号、指导教师、日期（6 字段，两列响应式）。姓名自动预填登录用户名；封面字段单向同步写回项目数据（触发自动保存，避免循环监听）。
- **就绪状态**：显示板 / 次梁 / 主梁是否已计算（✓/✗），未计算的章节以占位说明呈现。
- **A4 文档预览**：由 `ReportDocument.vue` 渲染 `buildReportDoc(data)` 装配的章节树。

### 八章节

| 章 | 内容 |
|----|------|
| 一、设计基本资料 | 结构尺寸、材料等级、荷载参数 |
| 二、结构平面布置及截面初估 | 布置说明、截面合理性 |
| 三、板设计 | 荷载 / 跨度 / 内力 / 正截面配筋，含公式推导、表、简图 |
| 四、次梁设计 | 含 T 形判断、斜截面 |
| 五、主梁设计 | 含集中力、吊筋 |
| 六、配筋结果汇总 | 三构件配筋汇总表 |
| 七、构造措施说明 | 钢筋构造要求 |
| 八、抵抗弯矩图说明 | Mu 图生成依据 |

- **公式渲染**：Unicode 数学符号（γd、ξ、αs、√、²、·、≤），不依赖 KaTeX；逐步代入实际数值。
- **打印**：点击「打印」调用 `window.print()`，`report-print.css` 控制 A4 分页（封面分页、表格 / 简图防跨页断开），暗色主题强制黑字白底。SVG 简图矢量打印。

> 文档模型 `ReportDoc{cover, sections[]}`，块类型 `heading/paragraph/formula/table/figure/note`；`figure` 用 `figureKind + props` 解耦，使装配器保持纯函数可单测。

## 8. 存档管理页（Archive）

**功能**：项目历史快照的版本管理。

- **项目卡片 + 时间线**：按项目分组，纵向时间线展示快照。
- **快照操作**：恢复（覆盖当前状态，二次确认）、派生 fork（从快照新建项目）、重命名、删除。
- **项目操作**：重命名、删除（二次确认，级联删除项目 + 全部快照）。

> 快照保存当时的完整 `data` JSON，可用于回溯「确认计算前的状态」或对比不同方案。

---

## 9. 系统设置页（Settings）

**功能**：个性化与账户管理。

- **主题切换**：暗 / 亮 / 暖三主题按钮组，立即生效，按用户分 `localStorage` 键持久化。
- **账户信息**：显示当前用户名。
- **注销账户**（危险区）：两次 ElMessageBox 确认，永久删除账户及全部数据（项目、快照、个人偏好），不可恢复。

> 主题系统实现（三主题令牌 + Element Plus `color-mix` 桥接）见 [技术架构 §主题系统](技术架构.md#三主题令牌系统)。
