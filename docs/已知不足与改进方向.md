# 已知不足与改进方向

> 对应任务书：九.11 已知不足和后续改进方向。

### 1. 主梁仅支持 3 跨

- **现象**：与板和次梁的完整实现任意跨数计算不同，主梁内力采用三等跨、三等分集中力系数表（表 14-4），系数表只覆盖 3 跨。
- **影响**：方案 C~F（主梁 4 / 5 跨）无法计算；本组方案 A 为 3 跨，已够用。
- **改进**：查阅规范手册补充 4、5 跨集中力系数表，或改用有限元 / 力法通用求解。

### 2. 抵抗弯矩图截断点

- **现象**：抵抗弯矩图（次梁）以 Mu 抗力带呈现，纵筋截断 / 弯起位置以文字说明。
- **原因**：求解器尚无配筋构造详图引擎（未输出各根纵筋的锚固 / 弯起 / 截断长度）。
- **改进**：扩展配筋构造模块，输出纵筋走向并绘制材料图。

### 3. 主梁弯矩剪力包络图

- **现象**：主梁当前以控制截面数值（M1、M2、MB、VA、VB 等）呈现最不利内力，未绘制连续包络曲线。
- **原因**：纯前端连不同工况极值易误导，需后端逐工况采样。
- **改进**：后端逐跨采样四工况内力，前端绘制连续包络图。
