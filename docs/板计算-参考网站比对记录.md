# 四、公式依据汇总

所有公式均来自 `docs/课设要求/` 中的课程文档：

## 荷载计算

**板：** 恒载标准值 = 水磨石面层 + 板厚×混凝土重度 + 抹灰厚×抹灰重度
**次梁：** 恒载标准值 = 板传来 + 自重 + 粉刷
**主梁：** 集中力 G/Q = 次梁支座反力 + 主梁自重折算 + 粉刷（直接输入）

## 荷载折算

- **板：** `g' = g + q/2`，`q' = q/2`
- **次梁：** `g' = g + q/4`，`q' = 3q/4`
- **主梁：** 集中荷载无需折算

## 内力计算

**板 / 次梁（均布荷载）：**
```
M = α·g'·l₀² + α₁·q'·l₀²
V = β·g'·lₙ + β₁·q'·lₙ
```
**主梁（三等分集中荷载）：**
```
M = K₁·G·l₀ + K₂·Q·l₀
V = K₁·G + K₂·Q
```
**支座边缘弯矩调整（次梁、主梁）：**
```
M' = |M_c| − b/2 × V₀
```

## 配筋计算

```
h₀ = h − c − d/2
αs = γd·M / (fc·b·h₀²)
ξ = 1 − √(1 − 2αs)
As = ξ·fc/fy·b·h₀
```

## 斜截面

```
Vc = 0.07·fc·b·h₀ / γd
Asv/s = γd·(V − Vc) / (1.25·fyv·h₀)
```
