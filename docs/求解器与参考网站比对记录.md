## 一、板计算

### 1.1 公式

**荷载：**
```
恒载标准值 qGk = 水磨石面层 + 板厚×混凝土重度 + 抹灰厚×抹灰重度
恒载设计值 qG = γG × qGk     (γG = 1.05)
活载设计值 qQ = γQ × qQk     (γQ = 1.20)
总荷载 q = (qG + qQ) × 1m
```

**跨度：**
```
中跨 l₀ = L / n          (L=18m, n=9跨实际 → l₀=2.0m)
边跨 l₀ = L/n − 0.120 + h/2   (参考网站简化：取2.0m，与中跨相同)
中跨 lₙ = L/n − b        (b=次梁宽200mm→0.2m, lₙ=1.8m)
边跨 lₙ = L/n − 0.120 − b/2   (参考网站简化：取1.8m)
```

**荷载折算（板）：**
```
g' = qG + qQ/2
q' = qQ/2
```

**内力（五跨连续等跨均布荷载系数）：**
```
M = α × g' × l₀² + α₁ × q' × l₀²
V = β × g' × lₙ  + β₁ × q' × lₙ
```

**配筋（正截面）：**
```
h₀ = h − c − d/2
αs = γd × M × 10⁶ / (fc × b × h₀²)
ξ = 1 − √(1 − 2αs)
As = ξ × (fc/fy) × b × h₀
```

## 二、次梁计算

### 2.1 公式

**荷载：**
```
板传来恒载   = qGk(板) × 次梁间距
次梁自重     = γ_concrete × b × (h − h_slab)
次梁粉刷     = γ_plaster × t_plaster × (h − h_slab) × 2
gk = 板传来 + 自重 + 粉刷
g  = γG × gk
qk = qQk(板) × 次梁间距
q  = γQ × qk
```

**跨度：**
```
弯矩 l₀: 边跨 = 中跨 = 7.2m（参考网站简化取轴线跨度）
剪力 lₙ: 边跨 = 7.2 − 0.24/2 − 0.25/2 = 6.955m
         中跨 = 7.2 − 0.25 = 6.950m
```

**荷载折算（次梁 — 与板不同）：**
```
g' = g + q/4      （板是 g' = g + q/2）
q' = 3q/4          （板是 q' = q/2）
```

**内力（五跨连续等跨均布荷载系数，与板相同）：**
```
M = α × g' × l₀² + α₁ × q' × l₀²
V = β × g' × lₙ  + β₁ × q' × lₙ
```

**支座边缘弯矩调整（次梁做了，板没做）：**
```
M' = |M_c| − b/2 × V₀
其中：
  b = 主梁宽 = 250mm = 0.25m
  V₀ = (g' + q') × l₀ / 2 = 简支梁剪力
  M' = 调整后的支座边缘弯矩（绝对值减小）
```

**配筋（正截面）：**
```
正弯矩（跨中）→ T形截面（第一类），宽度取 b_f = 2000mm = 次梁间距
负弯矩（支座）→ 矩形截面，宽度取 b = 200mm
h₀ = h − c − d/2 = 400 − 30 − 20/2 = 360mm
αs = γd·M / (fc·b·h₀²)
ξ = 1 − √(1 − 2αs)
As = ξ·(fc/fy)·b·h₀
     T形用 b_f=2000, 矩形用 b=200
```

**斜截面（箍筋）：**
```
Vc = 0.07·fc·b·h₀ / γd
Vsv = V − Vc
Asv/s = γd·Vsv / (1.25·fyv·h₀)
s  = Asv / (Asv/s)
```

## 三、主梁计算

### 3.1 公式

**荷载（集中力）：**
```
Gk = 次梁传来恒载 + 主梁自重 + 粉刷  (直接输入值求和)
G  = γG × Gk
Qk = 活载集中力                        (直接输入值)
Q  = γQ × Qk
```

**内力（三等跨连续梁，三等分集中荷载 — 课设表14-4）：**
```
4种荷载布置 → 取最不利组合包络
M = K₁ × G × l₀ + K₂ × Q × l₀
V = K₁ × G + K₂ × Q
```

**支座边缘弯矩调整：**
```
M' = |M_c| − b/2 × V₀
其中 b = 柱宽 = 350mm = 0.35m
     V₀ 取值约定待确认
```
