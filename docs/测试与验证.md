## 五、测试隔离与质量

- **数据库隔离**：`conftest.py::isolated_db` 使用 `tmp_path + monkeypatch` 重定向 `DB_PATH`，每个测试独立数据库，互不干扰。
- **API 集成**：FastAPI `TestClient`（httpx）直接打路由，覆盖鉴权门禁与异常路径。
- **超筋检测**：求解器核心 `common.py::xi()` 对 αs > 0.5 抛 `ValueError`（√ 负域），`/calculate` 捕获返回 400，`test_calculate_api.py` 验证。
- **纯函数优先**：抵抗弯矩、计算书装配、梁布局等均抽为纯函数并单测，与渲染解耦。

## 七、程序结果与手算结果对比

### 对比基准

求解器与参考网站 / 手算使用第 1 组方案 A 逐项比对。**总体：三构件荷载、内力、配筋选筋基本完全吻合。**

### 已知差异（均不影响配筋选筋结果）

| 项 | 差异来源 | 影响 |
|----|----------|------|
| 板边跨（M1、VA） | 程序用完整公式 l₀ = L/n − 0.120 + h/2，参考简化取 2.0 m | 板中间截面全吻合，边跨小差，不影响配筋 |
| 次梁支座弯矩 | 程序做支座边缘调整 M′ = |Mc| − b/2·V₀（V₀ 取简支剪力 63.22 kN） | 已对齐，Δ ≤ 0.05 |
| 主梁 MB 支座调整 V₀ 约定 | 参考用 V₀ ≈ 66.2 得 −218.67；本课程以教师为准取 V₀ = (G+Q)/2 得 −206.6862 | 差约 7.0 kN·m（3%），不影响配筋 |
| 斜截面 fc | 正截面用 fc = 9.6 吻合；参考斜截面疑似用旧规范 fc = 11 | 箍筋量小差，见 [已知不足](已知不足与改进方向.md) |

> 这些差异源于**公式约定 / 规范版本**而非计算错误，且不影响最终配筋选筋。程序以**教师平台数据**为最权威基准。

## 八、人工复核建议

教师 / 同学可按以下流程复核：

1. 运行 `pytest test_teacher_validation.py -v` 查看逐位断言；
2. 在程序中「填入演示参数」→ 确认计算，对照本文件第六节表格核对荷载 / 内力；
3. 任取一截面，按 [计算方法与公式](计算方法与公式.md) 手算 αs / ξ / As，与程序配筋表对照；
4. 斜截面若与手算有偏差，优先检查 fc（9.6 vs 11）与 h₀（h−c−d/2 vs h−c−d）约定。
