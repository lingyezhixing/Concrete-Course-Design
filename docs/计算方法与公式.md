> 对应任务书：九.7 主要公式与计算依据 / 九.5 程序输入参数说明。
> 本文档完整列出板、次梁、主梁的计算链、公式、系数表与工程约定，供答辩逐式说明与教师校核。

## 材料与系数常量

材料强度与分项系数在 [`backend/app/materials.py`](../backend/app/materials.py) **集中定义（单一来源）**，前端计算书镜像同一组常量（`frontend/src/report/materials.ts`），双侧同步。

| 符号 | 含义 | 取值 | 来源 |
|------|------|------|------|
| fc | C20 混凝土轴心抗压强度设计值 | **9.6 N/mm²** | SL 191-2008 |
| ft | C20 混凝土轴心抗拉强度设计值 | **1.10 N/mm²** | SL 191-2008 |
| fy（板筋） | Ⅰ级钢（HPB300）抗拉强度设计值 | **270 N/mm²** | 板受力筋 / 箍筋 / 构造筋 |
| fy（梁筋） | Ⅱ级钢（HRB335）抗拉强度设计值 | **300 N/mm²** | 梁纵向受力筋 |
| fyv | 箍筋抗拉强度设计值 | 270 N/mm² | Ⅰ级钢 |
| γd | 结构系数（钢筋混凝土 ψd） | **1.2** | SL 191-2008 |
| γG | 恒载分项系数 | **1.05** | 课程要求 |
| γQ | 活载分项系数 | **1.2** | 课程要求 |

## 计算流程总览

三构件统一遵循手算步骤：

```
参数派生 (derive.py) → 荷载计算 → 跨度确定 → 内力分析 → 正截面配筋 → 斜截面箍筋
```

- **派生**：扁平 `structure + loads` → 各构件 `Input`（荷载分量、跨度、截面），实现板 → 次梁 → 主梁荷载自动传递。
- **内力**：板、次梁用**等跨连续梁系数表**（均布荷载，2~5 跨，>5 跨按 5 跨简化）；主梁用**三等分集中力系数包络**（4 工况最不利）。
- **正截面**：统一 h₀ → αs → ξ → As，超筋（αs > 0.5）抛错由 API 返 400。
- **斜截面**：Vc = 系数·ft·b·h₀/γd，箍筋选型；主梁另算吊筋。

## 公共基本公式（`solvers/common.py`）

### 有效高度 h₀

```
h₀ = h − c − d/2
```
- h：截面高度（mm）；c：保护层厚度（mm）；d：估算纵筋直径（mm）。
- 默认：板 c = 20、d = 10；次梁 / 主梁 c = 30、d = 20。

### 正截面受弯（αs-ξ-As 法）

```
αs = γd·M / (fc·bw·h₀²)          （截面抵抗矩系数；M 单位 kN·m，余者 mm、N/mm²）
ξ  = 1 − √(1 − 2αs)               （相对受压区高度；αs > 0.5 时 √ 负域 → 超筋）
As = ξ·(fc/fy)·bw·h₀              （所需钢筋面积，mm²）
```
- bw：计算宽度。板取 1000 mm（1 m 板带）；次梁跨中 / 主梁按 T 形取翼缘宽 bf，支座按梁宽 b。
- ξ ≤ ξb（界限相对受压区高度），不满足判超筋。

### 配筋状态判定 `flexure_status`

```
As_提供 = 0           → "不足"
As_提供 / As_需要 > 1.8 → "建议复核"（超配，经济性提示）
其余                  → "推荐"
```

### 斜截面抗剪 `calc_shear_design`

混凝土抗剪承载力（均布荷载为主，系数 0.7）：

```
Vc = vc_coef · ft · b · h₀ / γd       （vc_coef：次梁 0.7、主梁 0.5）
```
- V ≤ Vc：构造配筋，间距 200 mm。
- V > Vc：`Asv/s = (γd·V − vc_coef·ft·b·h₀) / (fyv·h₀)`，间距 s 向下取整至 5 mm 且 ≤ 200 mm，配箍率 ρsv = Asv/(b·s)。
- 默认箍筋：次梁 Φ6、主梁 Φ10，双肢。

## 一、板（Slab）

板以次梁为支座，沿 L₂ 方向（width）连续，取 **1 m 板带** 计算。

### 1. 荷载（`slab/solver.py::calculate_load`）

恒载标准值（kN/m²）：

```
g_k = 水磨石面层 + 板厚 × 钢筋混凝土重度 + 抹灰厚 × 抹灰重度
g_d = γG · g_k = 1.05·g_k
q_d = γQ · 活载标准值 = 1.2·q_k
```

折算荷载（考虑支座整体连结，板取 1/2）：

```
g′ = g_d + q_d/2        q′ = q_d/2
```

### 2. 计算跨度（`calculate_spans` / `calculate_net_spans`）

```
中跨 l₀ = L/n
边跨 l₀ = L/n − 0.120 + h/2000          （0.120 m 为搁置长度，h 为板厚 mm）
净跨 l_n（中跨）= L/n − b                （b 为次梁宽）
净跨 l_n（边跨）= L/n − 0.120 − b/2
```

### 3. 内力（`common.py::calc_continuous_beam_internal_forces`）

采用 2~5 跨等跨连续梁系数表（均布荷载，[见下表](#等跨连续梁系数表-2-5-跨)）：

```
弯矩 M = α·g′·l₀² + α₁·q′·l₀²
剪力 V = β·g′·l_n + β₁·q′·l_n
```
>5 跨按 5 跨简化：首尾两跨用 1、2 跨系数，中间各跨均用第 3 跨系数。**板不做支座边缘调整**。

### 4. 正截面配筋（`slab/utils.py::calc_section_reinforcement`）

按 1 m 板带，b = 1000 mm，套用公共 αs-ξ-As 公式。按间距选筋：

- 直径候选 [6, 8, 10, 12, 14] mm，间距候选 [200, 180, 150, 120, 100, 80] mm；
- 每米提供面积 `As = π·d²/4 × (1000/s)`，选满足 As_需要 的最小候选；
- 表达形式 Φd@s（如 Φ8@200）。

### 5. 斜截面

板一般不需计算箍筋（由构造确定），程序不为板配置抗剪钢筋。

## 二、次梁（Beam）

### 1. 荷载（`beam/solver.py::calculate_beam_load`）

```
板传来恒载 = 板恒载标准值 × 次梁间距
次梁自重   = 钢筋混凝土重度 × 梁宽 × (梁高 − 板厚)
次梁粉刷   = 抹灰重度 × 抹灰厚 × (梁高 − 板厚) × 2
g_k = 板传来 + 自重 + 粉刷
g_d = γG · g_k
q_k = 活载标准值 × 次梁间距      q_d = γQ · q_k
```

折算荷载（主梁对次梁视为铰支，次梁取 1/4）：

```
g′ = g_d + q_d/4          q′ = 3·q_d/4
```

### 2. 计算跨度

```
净跨 l_n（边跨）= L − a/2 − b/2     （a 为次梁搁置长，b 为主梁宽）
净跨 l_n（中跨）= L − b
```
> 实现按轴线跨度作为 l₀（与参考网站简化一致）。

### 3. 内力

与板相同的等跨连续梁系数表。**做支座边缘弯矩调整**：

```
M′ = Mc + (b/2)·V₀          （Mc 为支座负弯矩，加正项使其绝对值减小）
V₀ = (g′ + q′)·l₀/2          （简支梁支座剪力）
```

### 4. 正截面配筋（`beam/utils.py::calc_beam_flexure`）

截面分类：

```
正弯矩（跨中，M ≥ 0）→ T 形截面
负弯矩（支座，M < 0）→ 矩形截面（宽取梁宽 b）
```

T 形类型判别：

```
Mu_limit = fc·bf·hf·(h₀ − hf/2) / γd
M ≤ Mu_limit → 第一类 T 形（计算宽 bw = bf = 次梁间距×1000）
M >  Mu_limit → 第二类 T 形（简化按 bw = bf 处理）
```

套用公共 αs-ξ-As 公式。按根数选筋：

- 直径 [12, 14, 16, 18, 20, 22, 25, 28, 32] mm，根数 [2~8]；
- 净距约束 `(b − n·d)/(n+1) ≥ 25 mm`（GB 50010）；
- 单筋面积表 Φ12=113.1 … Φ32=804.2 mm²，选满足 As 的最小候选，表达 nΦd（如 3Φ16）。

### 5. 斜截面箍筋（`beam/utils.py::calc_beam_shear` → `common.py`）

```
Vc = 0.7·ft·b·h₀/γd         （均布荷载为主，vc_coef = 0.7）
```
按公共抗剪流程选箍筋（默认 Φ6 双肢）。

### 6. 抵抗弯矩图（`composables/useResistingMoment.ts::computeMu`）

```
x  = fy·As / (fc·b)
Mu = fc·b·x·(h₀ − x/2)       （N·mm → kN·m）
```
逐跨以跨中截面 As 计算 Mu，在弯矩图上叠加 Mu 抗力带（绿色虚线）。

## 三、主梁（Main Beam）

主梁以柱为支座，承受次梁传来的**三等分集中力**，按 3 跨连续计算。

### 1. 荷载（`derive.py::derive_main_beam_input`）

每个次梁传来的集中力分量（kN）：

```
次梁恒载标准值 gk_beam = 板传来 + 次梁自重 + 次梁粉刷
from_beam_dead（次梁端反力）= gk_beam × 次梁跨度
self_weight（主梁自重）    = 钢筋混凝土重度 × 主梁宽 × (主梁高 − 板厚) × 次梁间距
plaster（主梁粉刷）        = 抹灰重度 × 抹灰厚 × (主梁高 − 板厚) × 次梁间距 × 2
live_load（活载）          = 楼面活载 × 次梁跨度 × 次梁间距

Gk = from_beam_dead + self_weight + plaster
G  = γG · Gk = 1.05·Gk
Q  = γQ · live_load = 1.2·live_load
```

### 2. 计算跨度（教师示例约定）

```
边跨跨中 M1 用   l₀_edge = 轴线跨度
中跨跨中 M2、支座 MB/MC 用  l₀_interior = 1.05 × (轴线跨度 − 柱宽)
```

### 3. 内力包络（`main_beam/utils.py::calc_envelope`）

采用三等跨、三等分集中力系数表（表 14-4，[见下表](#主梁集中力系数表-3-跨)）。活载 4 种最不利布置：

| 布置 | 活载位置 | 控制截面 |
|------|----------|----------|
| G_all | 全跨 | 恒载基础 |
| Q_13 | 1、3 跨 | M1、VA |
| Q_2 | 2 跨 | M2 |
| Q_12 | 1、2 跨 | MB、MC、VB左、VB右 |

```
弯矩 M = (K₁·G + K₂·Q)·l₀          剪力 V = K₁·G + K₂·Q
```
各截面取最不利组合（M1=G+Q_13，M2=G+Q_2，MB/MC=G+Q_12，VA=G+Q_13，VB左/VB右=G+Q_12）。

**支座边缘调整**：

```
M′ = Mc + (b_col/2)·V₀          V₀ = (G + Q)/2     （b_col 为柱宽）
```

### 4. 正截面配筋（`main_beam/utils.py::calc_main_beam_flexure`）

**课程简化**：所有截面（含支座）均按 T 形（bf = 次梁间距×1000）。套用公共 αs-ξ-As 公式，按根数选筋（同次梁）。

### 5. 斜截面箍筋与吊筋（`main_beam/utils.py::calc_main_beam_shear`）

```
Vc = 0.5·ft·b·h₀/γd           （集中荷载为主，vc_coef = 0.5，区别于次梁 0.7）
```

吊筋（次梁两侧）：

```
F   = from_beam_dead·γG + live_load·γQ     （次梁传来集中力设计值，不含主梁自重 / 粉刷）
Asb = γd·F / (fyv·sin45°)
```
